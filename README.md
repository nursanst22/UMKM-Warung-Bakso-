# UMKM-Warung-Bakso-
**Integrated Sales Information System - UMKM Warung Bakso**

This integrated Sales Information System is designed to transform the manual operations of a local meatball stall (UMKM Warung Bakso) into a centralized and efficient digital ecosystem.

**What the Project Does**
This project digitalizes the entire business cycle of the meatball stall, including daily sales transaction recording, raw material inventory management, and basic employee data administration. The system replaces manual ledgers with a relational database, ensuring that all business information is stored securely and structured systematically.

**Why the Project is Useful**
1. Data Accuracy: Minimizes the risk of recording errors commonly found in paper-based manual systems.
2. Automated Reporting: Enables the business owner to generate financial and stock reports immediately without performing manual recalculations.
3. Operational Efficiency: Accelerates the daily transaction process and simplifies the monitoring of raw material availability to prevent stock shortages.
4. System Integrity: Connects C# programming logic, SQL databases, and Crystal Reports into a single, seamless workflow.

**How Users Can Get Started**
To run this project in a local environment, follow these steps:
1.  Ensure your device has Visual Studio, SQL Server Management Studio (SSMS), and Crystal Reports runtime installed.
2.  Open SQL Server and create a new database,Import the database file provided in the repository folder.
3.   Adjust the connection string in the C# configuration file to link the application with your local SQL server.
4.   Open the solution file in Visual Studio, perform the build process, and run the application.
